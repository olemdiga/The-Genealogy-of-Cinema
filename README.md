# 🎬 The Genealogy of Cinema

A personal journey through film, framed by philosophy. This project analyzes cinema through the critical lenses of Nietzsche, Marx, Foucault, Fanon, Baudrillard, and other radical thinkers.

## ✨ Visit the site:

https://olemdiga.github.io/The-Genealogy-of-Cinema/

## 📖 About

The Genealogy of Cinema is a personal exploration of film that applies philosophical frameworks to uncover hidden structures of power, desire, ideology, and meaning. It's not academic criticism - it's a deep dive into how cinema reflects and shapes our understanding of the world.

## 🔑 Key Features

- **Philosophical Analysis:** Each film analyzed through a specific critical thinker's lens
- **0-10 Rating System:** Films rated for philosophical depth and critical application
- **Curated Organization:** 30+ film genres with thematic categorization
- **Critical Thinkers Directory:** Detailed profiles of 12+ philosophers and their conceptual frameworks
- **Refined Design:** Minimalist dark theme with thoughtful proportions and spacing
- **Mobile-First Experience:** Optimized for all devices with intuitive navigation

## 🎯 Philosophical Rating System

| Rating | Philosophical Depth | Description |
|---|---|---|
|10 | Masterpiece | Perfect philosophical vehicle, profound insights |
|8-9 | Excellent | Strong framework application, minor execution issues |
|6-7 | Good | Clear conceptual application, some theoretical gaps |
|4-5 | Average | Basic philosophical reference, superficial treatment |
|0-3 | Poor | Missed opportunities, weak or absent analysis |

## 🎨 Design Philosophy

The interface follows principles of **clarity, balance, and intention**:

- **Minimalist Dark Theme:** High-contrast typography on dark backgrounds
- **Thoughtful Proportions:** Balanced spacing and typographic hierarchy
- **Centered Content:** Main headings and key elements are center-aligned
- **Philosopher Color Coding:** Each thinker has a distinct accent color
- **Subtle Animations:** Purposeful transitions and hover states
- **Mobile-Optimized:** Touch-friendly with intuitive hamburger menu

## 🔧 Technical Implementation

- **Frontend:** Pure HTML5, CSS3, Vanilla JavaScript
- **Typography:** Inter font (Google Fonts) for modern readability
- **Icons:** Semantic emoji and Unicode symbols
- **Layout:** CSS Grid and Flexbox with responsive breakpoints
- **Animations:** CSS transitions and keyframe animations
- **No Dependencies:** Zero frameworks or build tools

## 📱 Mobile Experience

- **Hamburger Menu:** Clean overlay navigation
- **Touch-Optimized:** Larger tap targets, smooth scrolling
- **Responsive Grid:** Adapts from 4 columns down to 2 on mobile
- **Performance:** Optimized images and minimal JavaScript
- **Accessibility:** Semantic HTML and ARIA labels

## 🧠 Philosophical Frameworks

The project applies these critical lenses:

- **Nietzsche's Genealogy:** Tracing moral values to psychological origins
- **Marx's Historical Materialism:** Economics as base, culture as superstructure
- **Foucault's Archaeology of Knowledge:** Power through discourse and institutions
- **Fanon's Colonial Psychiatry:** Race as social construct and psychic wound
- **Baudrillard's Simulacra:** Hyperreality replacing the real
- **Mbembe's Necropolitics:** Sovereignty as power over life and death
- **Freud's Psychoanalysis:** The unconscious and dream-work in narrative
- **Butler's Performativity:** Gender as repeated performance
- **de Beauvoir's The Second Sex:** Woman as constructed Other
- **Adorno's Culture Industry:** Mass culture as manipulation
- **Schmitt's Political Theology:** Sovereignty and the state of exception
- **Habermas's Communicative Action:** Rational consensus and public sphere
